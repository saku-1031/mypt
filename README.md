# 漢字学習アプリ

日本語の漢字を効果的に学習するためのWebアプリケーションです。

## 機能

- 📚 **漢字一覧**: 学習する漢字のリスト表示
- 📖 **詳細情報**: 読み方、意味、例文の表示
- 🎴 **フラッシュカード**: インタラクティブな学習ツール
- 🎯 **クイズ**: 読み方と意味のテスト
- 📊 **進捗管理**: 学習した漢字の記録

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 本番環境で起動
npm start
```

## 使い方

1. サーバーを起動後、ブラウザで `http://localhost:3000` を開く
2. 漢字一覧から学習したい漢字を選択
3. フラッシュカードで学習、またはクイズに挑戦

## 技術スタック

- Backend: Node.js + Express + TypeScript
- Frontend: HTML/CSS/JavaScript
- Data: JSON
